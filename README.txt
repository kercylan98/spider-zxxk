# 已编译的包仅支持在 Linux 中运行（耗时极长，并且长时间高频行为下，内存及CPU占用率高，不建议个人电脑使用）
# 建议运行在没有特殊安全防御的设备，避免因为CPU占用率问题被误认为挖矿进程等

===== 说明 =======================================================

# 执行 start.sh 脚本即可开始爬取教材目录
# 执行 stop.sh 脚本可以优雅的停止教材目录爬取，避免数据写文件途中被破坏
# 执行 save.sh 脚本可以触发将结果保存到 result.xlsx 文件中的行为

# 执行 "tail -f xkw.log" 可以追踪爬取日志

===== 备注 =======================================================

# 程序在爬取每一层资源会优先从已有的缓存中进行建立，所以程序支持延续上一次的爬取进度

# data 目录中为数据缓存，当存在缓存的情况下不会再进行学科网的访问（极影响效率）
# 如果需要重新全面的爬取一次学科网教材目录，那么把 data 目录中的文件清空即可

# 程序默认仅会在正常结束的时候将结果进行导出，中途可以执行 save.sh 脚本进行导出
# 如果需要 Excel 结果，使用 save.sh 脚本后稍等片刻将 result.xlsx 下载即可

# 追踪看到的结果应该是乱序的，由于程序是在多线程情况下进行所致
